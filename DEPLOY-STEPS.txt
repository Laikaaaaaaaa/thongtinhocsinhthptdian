🚀 HƯỚNG DẪN DEPLOY THPT DĨ AN LÊN HEROKU
===============================================

BƯỚC 1: TẠO GITHUB REPOSITORY
-----------------------------
1. Mở trình duyệt đi tới: https://github.com/new
2. Repository name: thpt-di-an  
3. Description: THPT Dĩ An - Student Management System
4. Chọn "Public"
5. Nhấn "Create repository"

BƯỚC 2: PUSH CODE LÊN GITHUB
----------------------------
Chạy lệnh sau trong PowerShell:

cd "c:\Users\abc23\Desktop\CSDL-THPT DĨ AN"
git push -u origin main

BƯỚC 3: TẠO HEROKU APP
---------------------
1. Đi tới: https://dashboard.heroku.com/
2. Đăng nhập với: abc23072009@gmail.com
3. Nhấn "New" → "Create new app"
4. App name: thpt-di-an-2025
5. Region: United States  
6. Nhấn "Create app"

BƯỚC 4: KẾT NỐI GITHUB
----------------------
1. Trong Heroku app → tab "Deploy"
2. Deployment method: chọn "GitHub"
3. Nhấn "Connect to GitHub" và authorize
4. Repository name: thpt-di-an
5. Nhấn "Connect"

BƯỚC 5: THÊM DATABASE
--------------------
1. Tab "Resources"
2. Add-ons → tìm "Heroku Postgres"
3. Chọn plan "Essential" ($0/month)
4. Submit Order Form

BƯỚC 6: CẤU HÌNH ENVIRONMENT
---------------------------
Tab "Settings" → "Config Vars" → thêm:

SECRET_KEY: thpt-di-an-secret-key-2025-production
ADMIN_ACCOUNTS: abc23072009@gmail.com:admin123
SMTP_SERVER: smtp.gmail.com
SMTP_PORT: 587
SMTP_EMAIL: abc23072009@gmail.com
SMTP_PASSWORD: [Gmail App Password - xem hướng dẫn bên dưới]
FLASK_ENV: production

BƯỚC 7: DEPLOY
--------------
1. Tab "Deploy"
2. Manual deploy → chọn branch "main"
3. Nhấn "Deploy Branch"
4. Chờ build hoàn thành (~3-5 phút)

KẾT QUẢ
-------
✅ App URL: https://thpt-di-an-2025.herokuapp.com
✅ Admin: https://thpt-di-an-2025.herokuapp.com/admin

CÁCH TẠO GMAIL APP PASSWORD
---------------------------
1. Đi tới: https://myaccount.google.com/security
2. Bật "2-Step Verification" nếu chưa có
3. Vào "App passwords"
4. Chọn app: Mail
5. Generate password và copy
6. Dùng password này cho SMTP_PASSWORD
